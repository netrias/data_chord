Stage 2 — Passthrough icon variations

Goal: replace the misleading "0%" in the Value-fit column with iconography that conveys "this column is mapped to a standard with no permissible-value list, so values pass through unchanged." Direction settled on: arrow with a strikethrough in a neutral slate color, no enclosing prohibition-style red circle. This page surveys variations of that direction so we can pick.

Family A — Bare arrow + diagonal slash

Just the arrow with a slash through it — no enclosing badge or background. Most minimal. Each card shows the icon at row scale (left) and 2.2× scale (right) so you can compare line weight.

A1 — Thin straight, 45° slash
Lightest weight. Slash extends past the arrow on both sides.
A2 — Medium straight, 45° slash
Recommended baseline. Strong but not chunky.
A3 — Heavy straight, 45° slash
Bolder; reads more emphatic. Risk: starts feeling prohibition-y.
A4 — Short slash (only crosses arrow)
Quieter — slash sits within the arrow's footprint, not extending past it.
A5 — Vertical slash through arrow
Vertical line reads more like a "stop" than a "skipped" — tradeoff.
A6 — "Not equal" slash style
Steeper, narrower slash — borrows the "≠" gesture. Subtler.

Family B — Inside a soft badge

Same icon as A2, framed in a subtle background badge. Adds visual weight in the column without tipping into prohibition-sign territory (no red, no high-contrast border).

B1 — Gray-fill round badge
Soft gray-100 fill, gray-200 border. Reads as "informational chip".
B2 — Outline-only round badge
Hollow circle outline — quieter than B1, more emphasized than A2.
B3 — Square-rounded chip
Square chip with rounded corners — different visual language from B1/B2.

Family C — Color tone variations

Same shape as A2, exploring how light/dark slate or a touch of azure affects how the icon "weighs" in the column. None use red — staying clear of prohibition territory.

C1 — Light slate (gray-500)
Quietest. Reads as "informational, low-priority".
C2 — Mid slate (gray-700)
Recommended default — visible but not loud.
C3 — Dark slate (gray-900)
Heaviest neutral. Demands attention; might be too loud for an "informational" cell.
C4 — Azure tint
Borrows azure (already used for "override" elsewhere) — risk of visual collision.

Family E — Alternate negation gestures

Same arrow shape, but the negation is drawn differently. Worth comparing in case the diagonal slash doesn't feel right.

E1 — Dashed arrow (no slash)
Dashed arrow = "soft / non-binding flow". No explicit "no" gesture. Subtler.
E2 — Arrow with X overlay
Stronger negation. Risk: reads as "error / cancel".
E3 — Arrow with horizontal bar (block)
Hard vertical bar — reads as "blocked"; arguably wrong story.
E4 — Two parallel arrows (no slash)
No negation — just "flow continues". Clean alternative if strikethrough feels wrong.

Family F — Stronger arrow / slash contrast

Issue: when arrow and slash share the same stroke weight, they merge into an indistinct tangle and the negation gesture gets lost. These variants push the slash to read as a distinct mark on top of the arrow — via thicker stroke, halo cutout, filled bar, or color contrast. Each shows the icon at row scale and at 2.2× scale.

F1 — Slash 60% thicker than arrow
Slash stands forward as the dominant mark; arrow recedes to "subject being negated".
F2 — White-halo cutout slash
Slash visually punches through the arrow with a clean gap. Same technique used by Lucide's "ban" / Heroicons' "no-symbol". Cleanest professional read.
F3 — Heavy slash, light arrow
Color contrast separates the two. Arrow becomes a "ghosted" reference; slash carries the meaning.
F4 — Slash as filled bar
Solid bar reads as a deliberate "blocking" stamp. More architectural than F2's stroke approach.
F5 — True knockout (gap in arrow)
Arrow path is literally cut by the slash — no overlap at all. Most semantically literal "this arrow is broken".
F6 — Halo cutout + emphasized slash
F2's halo plus thicker slash. Strongest negation read while still feeling clean.

Family H — Bigger arrow + small X mark

Refined direction: the arrow dominates the icon (spans nearly edge-to-edge), and the negation is a small X sitting on the arrow shaft instead of a single slash. The X reads more explicitly as a "no" mark and, being smaller, doesn't fight the arrow for visual weight. Each card shows the icon at row scale (28px container) and 2.2× scale.

H1 — Big arrow + small X on shaft
X same weight as arrow but small enough not to compete. Reads as "marked / negated".
H2 — Big arrow + X with halo cutout
X clearly punches through the arrow shaft with a halo gap. Cleanest read of all H variants.
H3 — X at midpoint of arrow shaft
Smallest X yet — just a 3u mark. Quieter, more delicate.
H4 — X above the arrow shaft
X sits above (not on) the arrow — like a labeling mark. Cleaner separation but weaker negation.
H5 — Bigger arrow, X at arrowhead
X overlays the arrowhead — "the destination is negated". Different story from H1/H2.
H6 — Largest arrow, mid-sized X with halo
Maximum arrow + clear X with halo cutout. Boldest read; X is unambiguous.

Family J — H6 base, bigger X, color/opacity contrast

Building on H6: keep the big arrow, make the X bigger (~7u wide, was 5u), and introduce explicit visual separation between the two — via opacity ghosting, darker X stroke, or both. The arrow becomes the "subject" and the X is the dominant mark.

J1 — Arrow at 55% opacity, X full
Arrow becomes background context; X is unambiguously the main mark.
J2 — Arrow gray-400, X gray-900
Hard color contrast — arrow is "ghost-rail", X is decisive black mark.
J3 — Arrow gray-300, X gray-900
Lightest arrow + darkest X — strongest separation. Risk: arrow gets too faint to read.
J4 — Arrow gray-500 @ 65% opacity, X gray-900
Both color shift and opacity. Arrow stays readable; X clearly dominant.
J5 — Even bigger X (8u), stronger stroke
X grows to 8u wide, stroke 2.8 — boldest negation. Arrow still readable in gray-400.
J6 — Both slate-700, arrow @ 40%, X @ 100%
Same hue, just opacity — softer, more cohesive feel than the gray-900 X variants.

Family L — Fixed arrow + extreme weight contrast

Three fixes from the J4 feedback:

  • Arrow join — arrowhead is now a filled triangle, so the shaft butts cleanly into a solid head (no chevron-mouth gap).
  • Weight contrast — half the variants are thick arrow + thin X, half are thin arrow + thick X. Both directions tested.
  • Bigger X — 8-9u wide (was 7u), and positioned squarely on the shaft so the crossout gesture is unambiguous.
L1 — Thick arrow + thin X (stroked)
Heavy-bodied arrow, delicate ink-mark X. Reads as "thick arrow that has been crossed out".
L2 — Filled chunky arrow + thin X
Maximum-density arrow as a single filled shape, ultra-thin X on top. Most extreme thickness contrast.
L3 — Thin arrow + thick X
Faint outlined arrow + dominant heavy X. Reads as "an arrow has been crossed out".
L4 — Wireframe arrow + heavy X
Most extreme contrast in the L family. Wireframe arrow, X dominates. Risk: arrow might disappear at small sizes.
L5 — Thick filled + medium X (compromise)
Filled arrow body, mid-weight X — middle ground if both extremes feel wrong.
L6 — Thin arrow + huge X across full width
X crosses the entire arrow including the head. Most "decisive crossout" — clearly negates the whole shape.

Family N — Circle + arrow + slash (prohibition shape)

New direction: borrow the universal "no [thing]" pictogram — a circle containing the subject, with a diagonal slash through it. Same shape as the no-smoking sign, but in neutral slate (never red) so it reads as informational rather than as a warning. The shape itself does the heavy lifting: every viewer immediately parses "no X happening".

N1 — Outline circle, arrow, slash
All elements same weight, slate. The prohibition shape carries the meaning.
N2 — Slash with halo cutout
Slash punches through the circle and arrow with a clean halo gap. Most legible negation.
N3 — Filled circle, knockout arrow
Stamp-like: solid slate disc with white arrow knocked out, slash carved through. Boldest read.
N4 — Heavy circle, lighter arrow
Circle and slash are the structural elements (heavy slate); arrow is the "subject" (lighter). Hierarchy is explicit.
N5 — Slash inside circle only
Slash contained inside the circle — softer, less prohibition-y. Reads as "arrow with note attached" more than "forbidden".
N6 — Two-tone with halo
Light circle and arrow recede; slash dominates as the "no" gesture. Quietest informational read.

Family P — Off-the-shelf icons (raw, no composition)

Survey of existing free MIT-licensed icons we could use without composing. None of the major libraries (Lucide, Heroicons, Tabler, Phosphor, Bootstrap) ship a pre-built "arrow + circle + slash" combined icon — but they all have clean prohibition shapes. Worth considering whether a bare prohibition shape alone is enough in context (the tooltip carries the explanation; the arrow-inside is a nicety, not a necessity). All shown below are the exact unmodified SVGs from each library.

P1 — Lucide ban
Classic 45° slash extending outside the circle. The shape every viewer parses instantly. Lucide / MIT.
P2 — Tabler ban
Same shape as P1, very slightly different proportions (slash goes nearly corner-to-corner of circle). Tabler / MIT.
P3 — Lucide circle-slash
Slash contained inside the circle. Softer, less prohibition-y read. Lucide / MIT.
P4 — Tabler forbid
Tabler's contained-slash variant. Equivalent to P3, slightly different drawing. Tabler / MIT.
P5 — Heroicons no-symbol
Heroicons' take — slightly thinner stroke (1.5), slash exactly corner-to-corner of circle. Heroicons / MIT.
P6 — Phosphor prohibit
Phosphor's prohibit (256-grid). Slightly thicker visual feel. Phosphor / MIT.

Family Q — Library prohibition base + arrow inside

No library ships a combined "arrow + ban" icon, so for the compose path we use the library's circle/slash as our base and drop an arrow inside. Below: three of the cleanest library shapes from Family P, each with an arrow added.

Q1 — Lucide ban base + arrow
Lucide circle + their classic 45° slash + arrow inside. Slash uses halo cutout for clean separation.
Q2 — Tabler forbid base + arrow
Tabler base. Contained slash (doesn't extend past circle) — softer informational read.
Q3 — Heroicons base + arrow
Heroicons' lighter 1.5 stroke — feels more delicate. Slash spans full diameter of the circle.

What's actually available off-the-shelf

Available raw (no composition needed): Lucide ban, circle-slash, circle-slash-2; Tabler ban, forbid, forbid-2; Heroicons no-symbol; Phosphor prohibit; Bootstrap Icons slash-circle. All MIT-licensed.

Not available pre-built: a combined "arrow + ban" icon. Tabler has refresh-off, repeat-off, exchange-off which combine arrow loops + slash, but they encode "no recycle / no exchange" — wrong story for our passthrough case.

My pick: If the bare prohibition shape is enough (and the tooltip carries the explanation), go with P3 / P4 — Lucide circle-slash or Tabler forbid. The contained-slash variant is softer and more informational than the corner-to-corner ban. If you want the arrow inside, go with Q2 — Tabler forbid base with an arrow added; the contained slash leaves the arrow more legible than the corner-to-corner versions do. Either way you're using a battle-tested MIT-licensed shape as the foundation rather than hand-rolling.
