Stage 2 — Filter Button Style Explorations

Five base directions plus four combinations. Each bar is interactive — click a chip to see its active state. Counts and icons mirror the production code (✦ AI Recommendation, ✎ Override, ○ No Mapping). Token values are pulled from shared/static/tokens.css.

Baseline

V0 · Current production

baseline

What ships today: pill chips with neutral active state. Reference for comparison.

Active state is neutral gray — readable but doesn't reinforce the legend role.

Five base directions

V1 · Tinted-active chips

smallest lift

Same shape as today, but the active state adopts the category's color. The chip bar becomes the legend by construction.

Try: click "AI Recommendation" — netrias green confirms the filter and reinforces row icons.

V2 · Segmented control

single-select made obvious

One continuous pill with internal selection. Active segment lifts as a white card. The shape itself communicates "pick exactly one."

Modern, compact. Loses the per-category color reinforcement on its own — combine with V1 (see Combo A).

V3 · Underline tabs

data-toolbar feel

No frame. Active state is a 2 px colored underline; counts move into rounded pill badges. Reads like a GitHub Issues / Linear filter.

Lightest weight; great for "the table is the content." Sells the count badge as a first-class element.

V4 · Stat-card chips

numbers-first

Two-line tiles with the count promoted to the primary visual. The bar reads as a tiny dashboard — useful when users care "how many didn't map?".

Vertical real estate trade-off: bar is taller, but glanceable. Best when counts carry meaning.

V5 · Solid-active pill with badge counter

high-contrast selection

Inactive: ghost pill, count as plain text. Active: filled dark pill with a contrasting count badge. Notion / GitHub label vibe.

Strongest selection cue, but neutralizes category color. See Combo C for a tinted variant.

Combinations

Combo A · Segmented + tinted active

recommended

V2 + V1. Segmented shape says "pick one"; the tinted active state keeps the legend role. Compact, modern, semantic.

Try: tab through each option — the inner card lifts and tints in one motion.

Combo B · Underline tabs + stat counts

dashboard-ish

V3 + V4. Stat-card style numbers, but with the lighter underline-tab chrome. Fewer pixels than V4 yet still numbers-first.

Good middle ground: glanceable counts without the heavier chip frame.

Combo C · Solid-active pill, tinted

boldest

V5 + V1. Active fills with the category color (netrias green / azure / gray); icon stays inside a contrasting circle. Highest contrast option.

Most prominent — competes harder with the row data for attention. Useful if filters drive the workflow.

Combo D · Stat cards in a connected row

control panel

V4 fused into one rounded panel with internal dividers and a colored bottom accent on the active card. Reads as a single status board.

Most "reportable" — bar feels like a status board you'd screenshot for a stakeholder.
